NYPL Tech Connect

The New York Public Library's TechConnect program was created to meet an
increased demand for technology training and computer education. Our goal
is to help people of all ages learn how to use and develop their skills in
computers and technology in order to connect with family and friends,
advance their careers and expand their horizons.

TechConnect offers more than 80 technology classes at over 80 branch
libraries all absolutely FREE!

Locations include:

  • The Bronx
  • Manhattan
  • Staten Island

Project Code

The goal of the Project Code is to expose web programming to the diverse
audience served by the Library. Even if you do not pursue a career in web
development, learning how to code can teach you problem solving skills,
logical thinking, and how to think abstractly.

This program will help you learn basic web programming through a
blended learning model. We will be using online platforms (such as
Codecademy) and in class activities and exercises so you can get
practical handson experience.

    Project Code Phase I Focus:
  • Introduction to Coding Scripting
  • HTML and CSS
  • Website Layout Principles
    Project Code Phase II Focus:
  • JavaScript
  • JQuery and Libraries
  • BootStrap
